## Setterscript
Script that automatically sets up software config.

### Use
- set everything: `$ ./setterscript all`
- set bashrc: `$ ./setterscript bash`
- set emacs: `$ ./setterscript emac`
- set tmux: `$ ./setterscript tmux`
- set gdm3: `$ ./setterscript gdm3mon`
